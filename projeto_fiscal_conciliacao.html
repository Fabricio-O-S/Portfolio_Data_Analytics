Portal de Conciliação Fiscal 2026

Visão Geral do Projeto

Este projeto é um protótipo funcional para automação do processo de conciliação entre Notas Fiscais (NFs) e Pedidos de Compra (OCs), garantindo que o que foi faturado condiz com o que foi solicitado pelo ERP. Ele elimina a necessidade de conferência manual exaustiva, reduzindo erros e aumentando a velocidade do recebimento fiscal.

Principais Funcionalidades

  • Cruzamento Inteligente: Algoritmo em Python que valida chaves de acesso, itens e valores entre o XML da nota e o pedido de compra.
  • Processamento de Dados: Uso intenso da biblioteca Pandas para tratamento de grandes volumes de arquivos e cruzamento de bases.
  • Interface Intuitiva: Desenvolvido com Flask e Bootstrap, oferecendo feedback visual colorido para status de conciliação (Sucesso, Divergência ou Pendente).

Tecnologias Utilizadas

O "motor" de inteligência do projeto foi construído em Python, separando a lógica de negócio do servidor web (Flask). A estrutura de pastas segue os padrões de mercado (src, data, tests), garantindo um código limpo e fácil de manter.
