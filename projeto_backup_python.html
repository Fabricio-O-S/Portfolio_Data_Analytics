Sistema de Backup Profissional (Rclone + Python)

Visão Geral

Um gerenciador de backups profissional desenvolvido em Python com uma interface gráfica moderna. Este sistema simplifica a poderosa ferramenta de terminal Rclone, oferecendo um painel dinâmico para automatizar o fluxo de salvamento de arquivos em discos locais ou serviços de nuvem (Google Drive, OneDrive, etc.).

Diferenciais Técnicos

  • Feedback em Tempo Real: Uso de subprocess.Popen para monitorar o progresso da transferência bit a bit, atualizando a interface sem travamentos.
  • Fila Inteligente (Batch): Capacidade de processar múltiplas pastas de origens diferentes para um único destino de uma vez só.
  • Persistência de Dados (JSON): O sistema lembra das últimas configurações de caminhos utilizadas, facilitando o uso diário.
  • Segurança & Erros: Tratamento gracioso de erros, identificando bloqueios de antivírus ou permissões de pasta antes de falhar o processo.

Aprendizados e Evolução

Neste projeto, apliquei conceitos avançados de Threading para manter a interface responsiva durante processos pesados de I/O, além de explorar a biblioteca CustomTkinter para criar uma estética premium e profissional.

Resultados e Valor de Negócio

  • Segurança da Informação: Continuidade absoluta do negócio com recuperação rápida de dados em casos de desastre ou falha de hardware.
  • Eficiência de Custo: Otimização do uso de storage em nuvem através do motor Rclone, reduzindo custos de assinatura e tráfego.
  • Automação de Tarefas: Eliminação de processos manuais de cópia de arquivos, garantindo que nenhum dado crítico seja esquecido.
