# Portfólio de Data Analytics

## Descrição
Este repositório contém o portfólio de projetos e soluções em análise de dados, automação e infraestrutura, focado nas áreas de TI e Suprimentos. A arquitetura de base do portfólio segue uma interface web estruturada e limpa (HTML5/CSS), preparada para receber conteúdos detalhados de Python, PowerBI e bancos de dados.

## Como instalar e rodar
1. Não há dependências back-end iniciais (apenas HTML, CSS, JS).
2. Para visualizar o projeto, basta abrir o arquivo `index.html` em qualquer navegador web, ou hospedá-lo no GitHub Pages (ativando nas configurações do repositório).
3. Projetos futuros utilizando Python e bases de dados serão posicionados em `src/` e `data/`, documentados conforme necessário.

## O que aprendi com esse projeto
- Organização padronizada de arquivos seguindo as boas práticas Python e metodologias de desenvolvimento.
- Customização de temas HTML dinâmicos voltados para exposição de competências técnicas.
- Estruturação focada em "problema x solução" para demonstrar experiência prática e de negócios (exemplo: áreas de Logística, Tributária e Suprimentos).
